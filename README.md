## To Run Locally

Clone the project

```bash
  git clone https://github.com/mesbahi-01/GARDUINO.git
```

Go to the project directory

```bash
  cd GARDUINO_PROJECT
```
Go to the app directory

```bash
  cd app
```

Install the dependencies

```bash
  npm install
```
Start the React app 

```bash
  npm start
```

Go to the server directory

```bash
  cd ../server
```
Install the dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
